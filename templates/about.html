About This Project

This is a Python Flask web application with integrated AI chat capabilities, configured for deployment on Vercel.

Features:

  • Python Flask backend
  • Hugging Face DistilGPT-2 model (82M parameters)
  • Real-time AI chat without API keys
  • HTML templates with Jinja2
  • CSS styling with animations
  • API endpoints for data and chat
  • Vercel deployment ready

Tech Stack:

  • Flask 3.0.0
  • Transformers 4.36.0 (Hugging Face)
  • PyTorch 2.1.0
  • DistilGPT-2 LLM
  • HTML5 & CSS3
  • JavaScript (Fetch API)
  • Vercel Serverless Functions

AI Model:

  • Model: DistilGPT-2
  • Size: 82 million parameters
  • Type: Text generation
  • No API key required - runs directly on server

Made with Flask, Hugging Face & Vercel
